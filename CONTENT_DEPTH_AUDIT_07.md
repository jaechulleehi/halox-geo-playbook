# 07장 콘텐츠 깊이 보강 감사 메모

## 결론

07장은 산업별 페이지가 이미 나뉘어 있지만, 일부 페이지가 공통 체크리스트를 업종명만 바꿔 반복하는 상태였다. 이번 보강의 목표는 각 업종이 가진 고유한 질문 구조, 신뢰 근거, 전환 행동, 실패 리스크를 더 분명히 하는 것이다.

## 페이지별 보강 방향

| 페이지 | 보강한 핵심 |
|---|---|
| 07 parent | 산업별 GEO를 나누는 기준: 의사결정자/답변 단위/신뢰 출처/전환 행동/실패 리스크 |
| 07-01 B2B SaaS | 구매위원회 관점, 보안/연동/가격/사례 질문셋, 구매 질문별 콘텐츠 액션 |
| 07-02 커머스/플랫폼 | 상세페이지/Product schema/merchant feed/리뷰/정책의 상품 데이터 일치 기준 |
| 07-03 로컬/전문 서비스 | NAP/지도 프로필/리뷰/지점/전문성/외부 권위 신호와 지역 질문 액션 |
| 07-04 PR/뉴스룸 | 뉴스룸을 엔티티 허브로 보는 관점, 팩트시트/보도자료/바이오/FAQ 역할 |
| 07-05 금융/규제 | 노출보다 오해 방지, 최신성/과장/권한/비교/보안 리스크와 안전 표현 |
| 07-06 캠페인 URL | 캠페인 URL 유형별 역할, 종료 후 처리, D-7~D+30 측정 설계 |
