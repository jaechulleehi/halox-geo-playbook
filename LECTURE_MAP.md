## 강의 운영 원칙

- 매주 강의보다 산출물이 중요합니다.
- 한 주차의 메인 스토리는 하나만 둡니다.
- 심화 키워드는 본문 또는 토글/보충 자료로 분리합니다.
- HaloX 기능은 개념 설명 뒤의 분석/실습 단계에서 보여줍니다.
- 7장 산업별 전략과 8장 글로벌 GEO는 심화 세션 또는 후속 강의로 확장합니다.

## 1주차 상세 진행안

| 시간 | 진행 내용 | 산출물 |
|---|---|---|
| 사전 과제 | 00장 워크시트 작성 | GEO 한 문장 정의와 용어 비교표 |
| 1교시 | 01-01~01-04로 키워드를 질문으로 전환 | 키워드 10개, 질문 30개, 질문셋 포션표 |
| 2교시 | 01-05와 02장으로 baseline 측정 | mention/source/citation 초기 기록 |

1주차는 설명보다 산출물이 중요합니다. 작성 예시를 먼저 보여준 뒤 빈 워크시트로 전환하고, 마지막에는 2주차 fan-out에 넘길 대표 질문 3개를 고르게 합니다.


## 사례 기반 강의 확장안

기본 4주 강의는 유지하되, 각 주차마다 산업별 사례를 하나씩 끼워 넣으면 강의가 훨씬 살아납니다. 사례는 실제 미팅에서 얻은 문제 구조를 쓰되, 공개 강의에서는 익명 유형으로 바꿉니다.

| 주차 | 기본 실습 | 추가 사례 | 강의 효과 |
|---|---|---|---|
| 1주차 | 키워드에서 질문셋/Baseline 만들기 | 금융/거래소형 브랜드/비브랜드 질문 전환 | SEO 키워드와 AI 질문 시장의 차이가 선명해짐 |
| 2주차 | fan-out 질문맵/콘텐츠 갭 | 로컬 병원/지점형 지역 질문맵 | 질문 확장이 실제 지역/전환 문제와 연결됨 |
| 3주차 | 콘텐츠 리라이트/source map | PR 에이전시형/뉴스룸형 출처 설계 | GEO가 콘텐츠 구조와 평판 관리 문제임을 보여줌 |

## 4주차까지의 90분 수업 공통안

| 시간 | 진행 | 강사 확인 포인트 |
|---|---|---|
| 0~10분 | 과제와 오늘 산출물 확인 | 오늘 무엇을 제출할지 먼저 말한다 |
| 10~25분 | 핵심 개념과 사례 설명 | 설명은 산출물 작성에 필요한 만큼만 한다 |
| 25~55분 | 개인/팀 실습 | 워크시트가 비어 있지 않게 순회 피드백한다 |
| 55~75분 | 교차 피드백 | 질문/근거/액션 연결을 확인한다 |
| 75~85분 | 제출 템플릿 정리 | 다음 주차 입력값을 확정한다 |
| 85~90분 | 과제 안내 | 담당/기한/재측정 기준을 남긴다 |
