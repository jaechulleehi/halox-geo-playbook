# PDF_WORKBOOK

## 사용 방법

이 워크북은 4주 GEO 강의에서 수강생이 직접 작성할 실습지입니다. 각 주차의 목표는 하나의 산출물로 끝납니다.

- 1주차: 브랜드 baseline 표
- 2주차: Fan-out 질문맵과 콘텐츠 갭 리스트
- 3주차: 리라이트 초안과 source map
- 4주차: GEO 실행 리포트와 30일 액션 플랜
