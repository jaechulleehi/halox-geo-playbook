# 생성형 엔진 최적화(GEO) 실전 교과서: SEO에서 AI 검색 최적화까지

이 책은 AI 검색 시대에 브랜드가 ChatGPT, Perplexity, Gemini, Google AI Overviews 같은 답변형 검색 환경에서 어떻게 발견되고, 비교되고, 추천되고, 인용되는지 설명하는 실전형 GEO 플레이북입니다. 생성형 엔진 최적화(GEO), AI 검색 최적화, GEO 검색 최적화, LLM SEO/LLMO를 처음 접하는 사람이 개념을 잡고 실무 실행으로 옮길 수 있도록 구성했습니다.

기존 SEO가 검색 결과 페이지의 노출과 클릭을 다뤘다면, 생성형 엔진 최적화(GEO)는 AI 답변 안에서 브랜드가 어떤 질문에서 등장하고, 어떤 출처를 근거로 쓰이며, 어떤 경쟁사와 함께 검토되는지를 다룹니다. 이 책은 독자가 GEO 개념을 이해하고, 자기 브랜드의 질문셋/측정 지표/콘텐츠 갭/출처 전략/기술 점검을 직접 정리할 수 있도록 돕습니다.

## 대상 독자

- SEO/GEO를 함께 운영해야 하는 마케터
- AI 검색에서 브랜드 노출을 측정하고 싶은 브랜드/콘텐츠 팀
- 자기 브랜드의 AI 검색 상태를 점검하고 싶은 실무자
- GEO 도구/리포트/대행사 제안을 검토해야 하는 의사결정자

## 이 책의 사용법

1. 00장에서 생성형 엔진 최적화(GEO), SEO, AEO, AIO의 차이를 먼저 잡습니다.
2. 01~03장에서 SEO 키워드를 AI 질문 시장으로 바꾸고, 질문셋과 측정 기준을 만듭니다.
3. 04~06장에서 콘텐츠 구조, 답변 근거(source), 화면 인용(citation), AI 크롤러 접근성을 점검합니다.
4. 07~08장은 산업별/글로벌 상황에 맞춰 필요한 페이지만 골라 읽어도 됩니다.
5. 09장은 GEO 도구, 파트너, 리포트, 예산을 검토해야 할 때 참고합니다.
6. 10장은 앞에서 만든 기준을 4주 실행 로드맵과 GEO 리포트로 정리하는 흐름입니다.
7. 90장은 업종별 사례를 보고 싶을 때, 91장은 GEO 개념을 HaloX 분석 흐름으로 확인하고 싶을 때 참고합니다.

## 용어 사용 기준

시장에서는 AI 검색 최적화, LLM SEO, LLMO 같은 표현도 함께 쓰입니다. 이 책에서는 **생성형 엔진 최적화(GEO)**를 표준 용어로 사용하고, AEO/AIO/LLMO는 차이를 설명해야 할 때만 비교 개념으로 다룹니다.

또한 `source`는 **답변 근거**, `citation`은 **화면 인용**, `baseline`은 **기준선**, `crawler`는 **크롤러**로 풀어 씁니다. 처음 등장할 때만 영문을 병기하고, 이후에는 한국어 기준으로 읽히게 정리합니다.
