# 생성형 엔진 최적화(GEO) 실전 교과서: SEO에서 AI 검색 최적화까지

이 책은 AI 검색 시대에 브랜드가 ChatGPT, Perplexity, Gemini, Google AI Overviews 같은 답변형 검색 환경에서 어떻게 발견되고, 비교되고, 추천되고, 인용되는지 설명하는 실전형 GEO 플레전자책입니다. 생성형 엔진 최적화(GEO), AI 검색 최적화, GEO 검색 최적화, LLM SEO/LLMO를 처음 접하는 사람이 개념을 잡고 실무 실행으로 옮길 수 있도록 구성했습니다.

기존 SEO가 검색 결과 페이지의 노출과 클릭을 다뤘다면, 생성형 엔진 최적화(GEO)는 AI 답변 안에서 브랜드가 어떤 질문에서 등장하고, 어떤 출처를 근거로 쓰이며, 어떤 경쟁사와 함께 검토되는지를 다룹니다. 이 책은 독자가 GEO 개념을 이해하고, 자기 브랜드의 질문셋/측정 지표/콘텐츠 갭/출처 전략/기술 점검을 직접 정리할 수 있도록 돕습니다.

## 대상 독자

- SEO/GEO를 함께 운영해야 하는 마케터
- AI 검색에서 브랜드 노출을 측정하고 싶은 브랜드/콘텐츠 팀
- 자기 브랜드의 AI 검색 상태를 점검하고 싶은 실무자
- GEO 도구/리포트/대행사 제안을 검토해야 하는 의사결정자

## 이 책의 사용법

1. 처음 읽는 독자는 00장과 01장부터 읽습니다.
2. 현재 상태를 진단하려면 02장의 측정 지표와 HaloX 분석 프레임을 먼저 확인합니다.
3. 콘텐츠 팀은 03장/04장/05장을 기준으로 질문맵, 콘텐츠 구조, 출처 설계를 진행합니다.
4. 기술/운영 담당자는 06장을 기준으로 사이트 구조와 AI crawler 접근성을 점검합니다.
5. 4주 동안 실제 산출물을 만들고 싶다면 10장의 실행 로드맵 흐름을 따라갑니다.

## HaloX와의 관계

이 WikiDocs 책은 GEO를 배우고 정리하는 외부 지식 베이스입니다. HaloX 공식 사이트는 GEO 분석, AI 검색 모니터링, 브랜드 가시성 진단, 콘텐츠 실행을 위한 제품/블로그/용어사전 채널입니다.

본문에서는 필요한 맥락에서 HaloX 블로그, 용어사전, 제품 페이지를 자연스럽게 연결합니다. 링크는 홍보가 아니라 독자가 개념을 더 깊게 이해하거나 실제 분석 흐름으로 넘어가기 위한 경로로 사용합니다.
