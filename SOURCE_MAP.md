## 6. 세부 페이지 분화 계획

03장부터 10장까지는 장 소개 페이지와 세부 실행 페이지를 분리합니다. 장 소개 페이지는 독자가 흐름을 이해하는 입구로 두고, 세부 페이지는 강의자료/실무 체크리스트/이북 워크북으로 재사용할 수 있는 실행 단위로 관리합니다.

| 장 | 세부 페이지 방향 | 강의/이북 역할 |
|---|---|---|
| 03장 | Fan-out 질문맵, 질문 유형 포트폴리오, 콘텐츠 갭 | 2주차 질문 확장 실습 |
| 04장 | Answer-first, FAQ/표/schema, 리라이트 체크리스트 | 2~3주차 콘텐츠 개선 실습 |
| 05장 | Source/Citation 차이, Entity 합의 신호, 오프사이트 맵 | 3주차 출처 설계 실습 |
| 06장 | 기술 점검표, CSR/SSR, llms.txt/마이그레이션 | 4주차 테크니컬 점검 |
| 07장 | B2B SaaS, 커머스/플랫폼, 로컬/전문 서비스 | 심화 업종별 세션 |
| 08장 | 영문 카테고리 자산, locale/hreflang, 글로벌 source map | 글로벌 심화 세션 |
| 09장 | 대행사 검증, 도구/리포트 기준, 사업화 모델 | 의사결정자/사업화 세션 |
| 10장 | 1~4주차 워크숍과 실행 리포트 | 강의자료와 이북 워크북 골격 |


## 7. 사례 소스와 공개 전환 기준

고객사 미팅과 파트너 논의는 책의 강한 차별점이지만, 공개 WikiDocs와 이북에는 실명/계정/비공개 URL/구체 계약 맥락을 넣지 않습니다. 대신 산업별 익명 사례와 판단 기준으로 전환합니다.

| 사례 유형 | 원천 소스 | 공개 전환 방식 | 연결 장 |
|---|---|---|---|
| PR 에이전시형 | PR/콘텐츠 파트너 GEO 상품화 미팅 | `PR 에이전시가 GEO 진단 상품을 설계하는 사례`로 전환 | 05장/09장/10장 |
| 캠페인 URL 인용 추적형 | 캠페인/인플루언서 URL 모니터링 논의 | `캠페인 URL이 AI 답변에 반복 인용되는지 보는 사례`로 전환 | 02장/05장/07장 |
| 엔터프라이즈 뉴스룸형 | 대기업 커뮤니케이션/뉴스룸 운영 논의 | `뉴스룸/유튜브/온드미디어를 GEO 운영 체계로 관리하는 사례`로 전환 | 05장/07장/09장 |
| 로컬 병원/지점형 | 병원/오프라인 서비스 GEO 미팅 | `지점별 지역 질문과 내원 전환을 관리하는 사례`로 전환 | 07장/08장/10장 |
| 금융/거래소형 | 금융/거래소 SEO/GEO 미팅 | `브랜드 검색과 비브랜드 전환 키워드를 동시에 관리하는 사례`로 전환 | 01장/02장/06장/07장 |
| 커머스/플랫폼형 | 커머스 AIO 리서치/제품 노트 | `AI 고객이 상품을 비교/추천/구매 후보로 이해하는 사례`로 전환 | 07장/08장 |
| 글로벌/영문 확장형 | 글로벌 GEO 스터디/영문 readiness 노트 | `한국 브랜드가 영문 카테고리 자산과 외부 source map을 만드는 사례`로 전환 | 08장 |
| 대행사 검증형 | GEO 시장/대행사/사업화 스터디 | `GEO 대행사가 무엇을 증명해야 하는지 검증하는 사례`로 전환 | 09장 |

## 8. HaloX 기능/프레임 반영 기준

HaloX 기능은 광고처럼 먼저 노출하지 않고, 독자가 겪는 문제를 설명한 뒤 `이 문제를 분석할 때 어떤 화면/지표/워크플로우가 필요한가`로 연결합니다.

| 기능/프레임 | 독자 문제 | 책에서의 설명 위치 | 연결 장 |
|---|---|---|---|
| mention/source/citation 성과 프레임 | AI 답변에 언급되는 것과 실제 인용 근거가 되는 것을 구분하고 싶다 | 지표 정의/리포트 해석/워크시트 | 02장/05장 |
| 질문셋 포션/커버리지 | 몇 개 질문만 보고 성과를 판단해도 되는지 모르겠다 | 질문 시장과 baseline 진단 | 01장/02장/03장 |
| fan-out 질문맵 | 하나의 키워드가 AI 내부에서 어떤 질문으로 확장되는지 보고 싶다 | 질문맵/콘텐츠 갭 실습 | 03장 |
| AI 설명 품질 진단 | AI가 우리 브랜드를 어떤 문장으로 설명하는지 점검하고 싶다 | 브랜드 설명 품질/리라이트 기준 | 02장/04장 |
| AI 브리핑 인용 안정성 리포트 | 단발성 노출이 아니라 반복 인용 여부를 보고 싶다 | 인용 안정성/시계열 리포트 | 02장/10장 |
| 네이버/구글 AI 교차율 카드 | 검색 TOP10과 AI 브리핑/Overview의 차이를 보고 싶다 | 한국형 GEO 측정 프레임 | 02장/09장 |
| 병원 지점형 GEO 운영 | 지점별 지역 질문/URL/콘텐츠를 나눠 운영하고 싶다 | 로컬/전문 서비스 사례 | 07장 |
| 플랫폼 커머스 상세페이지 GEO 진단 | AI가 상품 상세를 추천 후보로 읽게 만들고 싶다 | 커머스/AIO 사례 | 07장 |
| 글로벌 영문 GEO readiness | 한국 브랜드가 영문 source와 디렉터리를 준비해야 하는지 판단하고 싶다 | 글로벌 확장 체크리스트 | 08장 |
| GEO 대행사 검증 체크리스트 | 대행사 제안이 진짜인지 포장인지 구분하고 싶다 | 대행사/도구/리포트 검증 | 09장 |
