## 이 책을 강의/실무/이북으로 활용하는 법

이 책은 강의자료, 실무 체크리스트, 이북 원고로 재활용할 수 있도록 설계합니다. 실무자는 먼저 [GEO 측정 지표와 HaloX 분석 프레임](https://wikidocs.net/346342)을 보고, 강의 운영자는 [4주 챌린지와 실행 리포트](https://wikidocs.net/346338)를 기준으로 재배열하면 됩니다.

## 세 가지 사용 방식

| 용도 | 읽는 흐름 | 산출물 |
|---|---|---|
| 실무 학습 | 00장부터 순서대로 읽기 | GEO 운영 기준 이해 |
| 4주 강의 | 01장/03장/05장/10장 중심 진행 | 주차별 실습 산출물 |
| 이북 제작 | 전체 장을 책 문장으로 재편집 | 판매용 PDF/EPUB |

## 강의로 쓸 때의 기준

강의에서는 모든 개념을 깊게 설명하려고 하면 흐름이 무너집니다. 매주 하나의 메인 스토리를 잡고, 심화 키워드는 보충 자료로 분리해야 합니다. 1주차는 baseline 진단, 2주차는 fan-out 질문맵, 3주차는 source/citation 설계, 4주차는 실행 리포트에 집중합니다.

## 사례로 이해하기

같은 WikiDocs라도 강의에서는 주차별 실습 흐름으로, 실무에서는 체크리스트로, 이북에서는 혼자 따라 하는 워크북으로 재배열해야 합니다.

## 실습 워크시트

| 입력 항목 | 작성 기준 |
|---|---|
| 활용 목적 | 강의/실무/이북 중 우선 목적 |
| 우선 읽을 장 | 00~10장 중 먼저 볼 장 |
| 필수 산출물 | 워크시트/리포트/액션 플랜 |
| 참고 링크 | HaloX 블로그/제품/용어집 중 연결할 자료 |
| 다음 일정 | 읽기/실습/리뷰 날짜 |

## 제출 템플릿

```text
활용 목적 / 우선 읽을 장 / 필수 산출물 / 참고 링크 / 다음 일정
```

## 작성 예시

아래 표는 “이 책을 강의/실무/이북으로 활용하는 법” 주제를 가상의 브랜드 상황에 적용한 예시입니다. 실제 작성 때는 자기 브랜드의 질문, 페이지, 출처 후보로 바꿔 적습니다.

| 입력 항목 | 작성 예시 |
|---|---|
| 활용 목적 | 4주 강의자료 |
| 우선 읽을 장 | 00~02장으로 개념과 baseline을 잡고 03~05장으로 실습 확장 |
| 필수 산출물 | 질문셋 30개, baseline 진단표, source map, 30일 액션 플랜 |
| 참고 링크 | HaloX GEO 개념 가이드와 AVI 점수 가이드 |
| 다음 일정 | 1주차 전까지 00~02장 워크시트를 채운다 |

## 완료 기준

- 강의/실무/이북 중 어떤 방식으로 읽을지 정해졌습니다.
- 각 장에서 만들어야 할 산출물이 보입니다.
- 4주 챌린지로 옮길 준비가 됩니다.
