## 질문셋 포션이 중요한 이유

질문셋 포션은 GEO 측정의 품질을 좌우합니다. 포션을 정한 뒤에는 [질문셋에서 콘텐츠 갭을 찾는 법](https://wikidocs.net/346346)으로 실행 과제를 나눕니다.

질문셋 포션은 우리가 측정하는 질문 묶음이 전체 시장 질문 중 어떤 영역을 대표하는지 보는 개념입니다. 브랜드 질문만 측정하면 비브랜드 추천 문맥을 놓칠 수 있고, 정보형 질문만 측정하면 비교/구매 문맥을 놓칠 수 있습니다.

## 질문셋을 나누는 방식

| 질문군 | 예시 | 의미 |
|---|---|---|
| 브랜드형 | HaloX는 어떤 도구야? | 이미 브랜드를 아는 사람의 질문 |
| 카테고리형 | GEO 분석 도구는 무엇이 있나? | 시장 탐색 질문 |
| 비교형 | HaloX와 Semrush를 비교해줘 | 검토 단계 질문 |
| 추천형 | B2B SaaS에 맞는 AI 검색 모니터링 도구 추천해줘 | 후보 선정 질문 |
| 실행형 | AI 검색 노출을 높이려면 무엇부터 해야 하나? | 실행 전환 질문 |

## HaloX 분석과 연결되는 지점

질문셋 포션을 정하면 HaloX 리포트에서 어떤 질문군을 우선 볼지 결정할 수 있습니다. 단순 언급률보다 질문군별 언급률, 경쟁사 동시 언급률, 선택 이유, source/citation 패턴을 함께 보는 것이 중요합니다.

## 사례로 이해하기

질문셋 포션은 측정의 표본 설계입니다. 브랜드형 질문만 넣으면 좋아 보일 수 있지만, 비브랜드 추천형 질문에서 빠지는 문제는 놓칩니다.

## 실습 워크시트

| 입력 항목 | 작성 기준 |
|---|---|
| 질문군 | 브랜드형/카테고리형/비교형/추천형/실행형 |
| 질문 수 | 전체 질문셋에서 몇 개를 넣을지 |
| 사업 의미 | 인지/검토/구매/실행 중 어떤 단계인지 |
| 우선순위 | 이번 baseline에서 반드시 봐야 하는지 |
| 다음 액션 | 질문 추가/삭제/재분류 |

## 제출 템플릿

```text
질문군별 포션표 / 우선 질문군 / 제외할 질문 / 다음 측정 기준
```

## 작성 예시

아래 예시는 `AcmeGEO`라는 가상의 B2B SaaS 브랜드를 기준으로 작성한 샘플입니다. 실제 강의에서는 브랜드명, 질문, 출처 후보, 담당자를 자기 상황에 맞게 바꿉니다.

| 입력 항목 | 작성 예시 |
|---|---|
| 브랜드형 | 5개, AcmeGEO를 이미 아는 사용자의 질문 |
| 카테고리형 | 8개, GEO 분석 도구 시장 탐색 |
| 비교형 | 7개, AcmeGEO와 경쟁사 비교 |
| 추천형 | 7개, B2B SaaS 조건 추천 |
| 실행형 | 3개, 도입 후 측정 절차 확인 |

## 완료 기준

- 개념 설명에서 끝나지 않고 다음 측정/작성/실행 액션이 보입니다.
- 질문, 콘텐츠, 출처, 측정 기준이 서로 연결되어 있습니다.
- 강의 후 제출물로 쓰거나 이북 워크시트로 옮겨도 설명이 부족하지 않습니다.
