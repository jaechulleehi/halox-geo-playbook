## 1주차 실습: 내 브랜드 GEO Baseline 진단

1주차 실습의 목표는 우리 브랜드의 현재 GEO 상태를 진단하는 것입니다. 진단 결과는 [1주차: GEO Baseline 진단 워크숍](https://wikidocs.net/346365)의 산출물로 정리합니다.

## 오늘 만들 산출물

- 핵심 키워드 10개
- AI 질문/프롬프트 30개
- 주요 경쟁사 3곳
- 질문군별 브랜드 언급 상태
- mention/source/citation 초기 기록
- 다음 주에 확장할 대표 질문 3개

## 진행 순서

1. 브랜드/서비스의 핵심 키워드 10개를 고릅니다.
2. 각 키워드를 정보 탐색형/비교형/추천형/검증형 질문으로 바꿉니다.
3. 주요 경쟁사 3곳을 정합니다.
4. HaloX에서 브랜드와 경쟁사를 기준으로 질문셋을 입력합니다.
5. AI 답변에서 브랜드가 어떻게 보이는지 기록합니다.
6. 다음 주 fan-out 분석에 사용할 대표 질문 3개를 고릅니다.

## 체크리스트

- 브랜드명 질문만 넣지 않았는가?
- 비브랜드 추천형 질문이 포함됐는가?
- 비교형 질문이 포함됐는가?
- source와 citation을 구분해서 기록했는가?
- 다음 주에 확장할 질문을 3개 이하로 좁혔는가?

## 사례로 이해하기

1주차 baseline은 잘하려는 보고서가 아니라 현재 위치를 정확히 보는 출발선입니다. 점수가 낮아도 괜찮지만, 어떤 질문에서 왜 낮은지는 남겨야 합니다.

## 실습 워크시트

| 입력 항목 | 작성 기준 |
|---|---|
| 브랜드명 | 측정할 브랜드 |
| 질문셋 | 포션별 질문 30개 |
| 경쟁사 | 함께 비교할 후보 3곳 |
| 측정 지표 | mention/source/citation/co-mention/answer quality |
| 다음 주 과제 | Fan-out으로 확장할 대표 질문 3개 |

## 제출 템플릿

```text
브랜드명 / 질문셋 30개 / 경쟁사 3곳 / baseline 기록표 / 다음 주 대표 질문 3개
```

## 작성 예시

아래 예시는 `AcmeGEO`라는 가상의 B2B SaaS 브랜드를 기준으로 작성한 샘플입니다. 실제 강의에서는 브랜드명, 질문, 출처 후보, 담당자를 자기 상황에 맞게 바꿉니다.

| 입력 항목 | 작성 예시 |
|---|---|
| 브랜드명 | AcmeGEO |
| 질문셋 | 브랜드형 5개, 카테고리형 8개, 비교형 7개, 추천형 7개, 실행형 3개 |
| 경쟁사 | Semrush, Ahrefs, Profound |
| 측정 결과 | 30개 중 mention 4개, citation 0개, 추천형 질문에서 경쟁사 2곳 반복 등장 |
| 다음 주 과제 | GEO 도구 추천 / AI visibility 측정법 / GEO 리포트 해석법 |

## 완료 기준

- 개념 설명에서 끝나지 않고 다음 측정/작성/실행 액션이 보입니다.
- 질문, 콘텐츠, 출처, 측정 기준이 서로 연결되어 있습니다.
- 강의 후 제출물로 쓰거나 이북 워크시트로 옮겨도 설명이 부족하지 않습니다.
