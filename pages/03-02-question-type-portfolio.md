## 질문 유형 포트폴리오를 어떻게 나눌까

GEO 질문셋은 정보형 질문만으로 채우면 안 됩니다. 비교형, 추천형, 검증형, 실행형 질문이 함께 있어야 AI 검색에서 실제 의사결정 문맥을 잡을 수 있습니다.

질문 포트폴리오가 한쪽으로 쏠리면 측정 결과도 왜곡됩니다. 정보형 질문에서는 보이지만 구매/추천 질문에서는 사라지는 브랜드가 많습니다.

## 실무에서 먼저 볼 것

| 점검 항목 | 확인 질문 | 다음 액션 |
|---|---|---|
| 질문 | 이 페이지가 답하는 핵심 질문은 무엇인가? | 질문을 한 문장으로 다시 쓴다 |
| 자산 | 이미 답할 콘텐츠나 페이지가 있는가? | 신규/리라이트/보강으로 나눈다 |
| 출처 | AI가 참고할 근거가 충분한가? | 내부 근거와 외부 근거를 분리한다 |
| 측정 | 어떤 지표로 변화를 볼 것인가? | mention/source/citation/co-mention 중 핵심 지표를 고른다 |

## 실행 순서

1. 현재 질문 목록을 한곳에 모읍니다.
2. 질문을 정의형/비교형/추천형/검증형/실행형으로 표시합니다.
3. 특정 유형에 몰린 질문을 찾아 균형을 맞춥니다.
4. 구매 전환에 가까운 질문을 따로 표시합니다.
5. 콘텐츠 제작 순서를 질문 유형별로 정합니다.

## 사례로 이해하기

질문셋이 전부 `무엇인가`형이면 AI 답변은 정의에는 잡히지만 추천/비교 문맥에서는 빠집니다. 강의에서는 같은 주제를 정의형, 비교형, 대안형, 가격/도입형, 실패 방지형으로 나눠 질문 포트폴리오를 만듭니다.

## 실습 워크시트

| 입력 항목 | 작성 기준 |
|---|---|
| 질문 유형 | 정의/비교/추천/검증/실행 |
| 대표 질문 | 사용자 말투로 작성 |
| 구매 단계 | 문제 인식/비교/도입/검증 |
| 필요 근거 | 사례/수치/비교표/FAQ |
| 우선순위 | 이번 주에 처리할 질문 |

## 제출 템플릿

```text
카테고리 / 질문 유형별 5개 질문 / 구매 단계 / 필요한 근거 / 담당 콘텐츠 / 측정 지표
```

## 작성 예시

아래 표는 “질문 유형 포트폴리오를 어떻게 나눌까” 주제를 가상의 브랜드 상황에 적용한 예시입니다. 실제 작성 때는 자기 브랜드의 질문, 페이지, 출처 후보로 바꿔 적습니다.

| 입력 항목 | 작성 예시 |
|---|---|
| 정의형 | GEO 모니터링이란 무엇인가? |
| 비교형 | GEO 도구와 SEO 도구는 무엇이 다른가? |
| 추천형 | B2B SaaS에 적합한 GEO 분석 도구는? |
| 검증형 | AI 답변에 나온 브랜드 언급은 어떻게 검증하나? |
| 실행형 | 우리 브랜드의 AI visibility를 30일 동안 개선하려면 무엇부터 해야 하나? |

## 완료 기준

- “질문 유형 포트폴리오를 어떻게 나눌까” 실습 결과가 질문맵으로 남습니다.
- 질문별 콘텐츠/출처 빈칸이 분리됩니다.
- 다음 실습에서 처리할 우선 질문이 정해집니다.

## 참고 링크 패키지

질문 유형을 나눌 때 막히면 HaloX의 [SEO/GEO 키워드 전략 프레임워크](https://haloxlabs.ai/ko/blog/seo-geo-keyword-strategy-framework)를 참고합니다. 키워드 하나가 정의형/비교형/추천형 질문으로 갈라지는 방식을 이해한 뒤, 이 페이지의 포트폴리오 표에 자기 브랜드 질문을 배치하면 됩니다.

## 체크리스트

- 질문 유형이 한쪽으로 몰리지 않았는가?
- 구매 전환에 가까운 질문을 따로 표시했는가?
- 정의형 질문만 많고 비교/검증 질문이 부족하지 않은가?
- 콘텐츠 제작 순서가 질문 유형과 연결되는가?
- baseline 측정에 넣을 질문이 남았는가?

## 흔한 질문

**Q. 이 페이지의 내용을 바로 강의자료로 써도 되나요?**

가능합니다. 다만 강의에서는 설명보다 실습 산출물을 먼저 정해야 합니다.

**Q. 모든 항목을 한 번에 실행해야 하나요?**

아닙니다. 질문 유형의 균형을 먼저 맞춘 뒤, 구매/상담에 가까운 질문부터 콘텐츠 과제로 옮기면 됩니다.
