## 엔티티와 브랜드 합의 신호를 만드는 법

![엔티티와 브랜드 합의 신호](../assets/images/page-heroes/halox-geo-05-02-entity-consensus-offsite-hero.png)

엔티티는 AI와 검색엔진이 “이 브랜드가 무엇이고, 어떤 주제와 연결되는지” 이해하기 위한 식별 단위입니다. 브랜드 합의 신호는 공식 사이트, 외부 프로필, 언론, 커뮤니티, 디렉터리에서 같은 핵심 설명이 반복되는 상태입니다.

합의 신호는 같은 문장을 복사해 뿌리는 일이 아닙니다. 채널마다 문장은 달라도 카테고리, 대상 고객, 핵심 기능, 신뢰 근거가 충돌하지 않아야 합니다.

## 공식 사이트에서 시작한다

외부 신호를 만들기 전에 공식 사이트의 기준 문장을 먼저 정합니다. About, 제품 페이지, FAQ, schema, 회사 프로필에서 브랜드를 다르게 설명하면 외부 출처가 늘어도 AI가 일관되게 이해하기 어렵습니다.

| 필드 | 예시 |
|---|---|
| 브랜드명 | AcmeGEO |
| 카테고리 | AI 검색 브랜드 가시성 분석 도구 |
| 대상 | B2B SaaS 마케팅/콘텐츠 팀 |
| 핵심 지표 | mention/source/citation |
| 대표 산출물 | GEO 리포트와 30일 액션 플랜 |

## 합의 신호는 반복성과 일관성이다

브랜드 설명이 여러 곳에 흩어져 있어도 같은 방향을 가리키면 좋은 신호가 됩니다. 반대로 한 곳에서는 “SEO 도구”, 다른 곳에서는 “AI 콘텐츠 생성기”, 또 다른 곳에서는 “마케팅 자동화 플랫폼”이라고만 쓰이면 AI가 브랜드를 어떤 카테고리로 이해해야 할지 헷갈립니다.

특히 GEO에서는 비브랜드 질문에서 어떤 카테고리 후보로 들어가는지가 핵심입니다. 브랜드명보다 문제/카테고리/지표 언어가 먼저 정리돼야 합니다.

![브랜드 엔티티 합의 신호 지도](../assets/images/body-figures/halox-geo-05-02-entity-consensus-constellation-codex-only.png)

*합의 신호는 여러 채널의 설명이 같은 브랜드 정체성을 가리키는 상태다.*

## 채널별로 확인할 것

- 공식 사이트: 카테고리와 핵심 지표가 첫 화면에서 보이는가
- 언론/PR: 브랜드가 어떤 문제를 해결한다고 설명되는가
- 위키/디렉터리: 이름, 설명, URL, 창업자/조직 정보가 일관적인가
- 외부 블로그: 같은 주제의 질문에 같은 기준으로 설명되는가
- 커뮤니티: 실제 사용 맥락과 오해가 무엇인지 보이는가

## AcmeGEO 예시

AcmeGEO의 공식 사이트가 “AI 검색 브랜드 가시성 분석 도구”라고 말하는데, 외부 글에서는 “SEO 순위 추적기”라고만 소개된다면 합의 신호가 약합니다. 이 경우 외부 글을 무작정 늘리기보다 공식 설명, 리포트 예시, 비교 문서, 프로필 정보를 먼저 맞춥니다.

그다음 언론/외부 블로그/디렉터리에서 같은 방향의 설명이 쌓이도록 관리합니다.

## 정리 양식

```text
브랜드 기준 문장:
핵심 카테고리:
대상 고객:
반복되어야 할 지표/기능:
충돌하는 외부 설명:
수정할 공식 페이지:
보강할 외부 채널:
```
