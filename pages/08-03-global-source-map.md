## 글로벌 source map: 시장별 답변 근거 설계

![글로벌 source map](../assets/images/page-heroes/halox-geo-08-03-global-source-map-hero.png)

글로벌 source map은 해외 백링크 목록이 아닙니다. 시장별로 AI가 신뢰할 만한 공식 문서, 미디어, 디렉터리, 커뮤니티, 비교 글을 질문군 기준으로 정리하는 지도입니다.

같은 브랜드도 미국, 일본, 한국에서 신뢰하는 출처가 다를 수 있습니다. 글로벌 GEO는 시장별 질문과 source 후보를 분리해야 합니다.

## 먼저 볼 기준

| 기준 | 읽는 법 |
|---|---|
| 시장 | 국가/언어별 질문군을 따로 만든다 |
| 출처 | 미디어/디렉터리/커뮤니티/공식문서를 나눠 본다 |
| 우선순위 | 비브랜드 추천형 질문에 영향을 주는 출처부터 본다 |

## 실행 흐름

1. 목표 시장의 대표 질문과 현지 source 후보를 나눈다.
2. AI 답변에서 현지 언론, 리뷰, 디렉터리, 경쟁 URL을 분리한다.
3. 우리 공식 영문 자산과 외부 source 사이의 공백을 표시한다.
4. 현지 카테고리 설명, 비교 근거, 프로필 정보를 먼저 보강한다.
5. 같은 시장 질문으로 source/citation 구성이 바뀌었는지 확인한다.

## 글로벌 source 예시

AcmeGEO가 영어권에 진입할 때는 한국어 기사보다 영어 제품 설명, 글로벌 SaaS 디렉터리, 비교 글, 영문 리포트 예시가 더 직접적인 source 후보가 됩니다. 시장별로 필요한 근거가 다릅니다.

```text
재측정 질문:
```
