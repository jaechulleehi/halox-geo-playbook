## GEO 대행사는 무엇을 증명해야 하나

좋은 GEO 대행사는 “AI 검색에 노출시켜드립니다”가 아니라 어떤 질문에서 브랜드가 빠지고, 어떤 출처가 부족하며, 어떤 콘텐츠/기술/오프사이트 과제를 실행해야 하는지 증명해야 합니다. 이름만 GEO인 제안과 실제 운영 가능한 제안은 질문셋과 재측정 계획에서 갈립니다.

대행사를 평가할 때는 화려한 용어보다 운영 증거를 봐야 합니다. 질문 설계, baseline 측정, source/citation 해석, 콘텐츠 액션, 기술 점검, 월간 리포트가 하나의 흐름으로 연결되어야 합니다.

## 대행사 제안서에서 확인할 6가지

| 검증 항목 | 좋은 답변 | 위험 신호 |
|---|---|---|
| 질문셋 | 브랜드/산업/구매 단계별 질문을 나눠 제시 | “AI에 많이 나오게”처럼 범위가 모호함 |
| 측정 지표 | mention/source/citation/경쟁사/변동성을 분리 | 단일 점수만 제시 |
| 콘텐츠 액션 | 신규 작성/리라이트/FAQ/표/schema를 구분 | 블로그 글 수만 제안 |
| 오프사이트 | 언론/리뷰/커뮤니티/파트너 출처를 분리 | 백링크 수만 강조 |
| 테크니컬 | robots/sitemap/rendering/schema를 점검 | 개발 이슈를 전혀 다루지 않음 |
| 재측정 | 같은 질문셋으로 전후 비교 | 결과 기준 없이 월간 보고만 약속 |

## 상품 유형별로 물어볼 질문

| 상품 유형 | 꼭 물어볼 질문 | 받아야 할 산출물 |
|---|---|---|
| 진단형 | 지금 어떤 질문에서 빠져 있나 | baseline 리포트/질문셋/우선순위 |
| 전략형 | 무엇부터 고쳐야 하나 | 30일 실행 로드맵/source map |
| 운영형 | 매월 무엇을 반복하나 | 월간 리포트/콘텐츠 수정안/재측정표 |
| 교육형 | 팀이 무엇을 직접 할 수 있게 되나 | 워크시트/제출물/피드백 기준 |
| 제품형 | 도구가 어떤 의사결정을 줄이나 | 대시보드/알림/리포트 export |

## 사례로 이해하기

PR 에이전시형 고객은 “우리 고객사가 AI 답변에 잘 보이게 해주세요”라고 요청할 수 있습니다. 이때 좋은 대행사는 보도자료 배포 횟수를 먼저 말하지 않습니다. 브랜드가 어떤 질문에서 언급되는지, 어떤 매체가 source로 잡히는지, citation으로 실제 링크되는 페이지가 무엇인지, 부정확한 답변이 반복되는지부터 봅니다.

캠페인 URL 인용 추적형 고객은 특정 랜딩페이지가 AI 답변에 인용되는지 확인하고 싶어합니다. 이 경우에는 질문셋, 캠페인 URL, 대체로 인용되는 경쟁 페이지, URL 구조, redirect/canonical 상태까지 함께 봐야 합니다. 단순 노출 리포트로는 충분하지 않습니다.

## 대행사 선정 질문 예시

```text
1. 우리 산업에서 측정할 질문셋은 누가 어떻게 만드나요?
2. mention/source/citation을 각각 어떻게 구분하나요?
3. AI 답변에서 경쟁사가 나오는 이유를 어떻게 설명하나요?
4. 콘텐츠 수정과 오프사이트 보강 중 무엇을 먼저 하라고 판단하나요?
5. robots/sitemap/rendering/schema 같은 기술 문제도 리포트에 포함되나요?
6. 4주 뒤 같은 질문셋으로 무엇을 비교하나요?
```

## 실습 워크시트

| 입력 항목 | 작성 기준 |
|---|---|
| 검증 항목 | 질문셋/측정/콘텐츠/출처/기술/리포트 |
| 대행사 답변 | 제안서나 미팅에서 확인한 내용 |
| 증거 | 샘플 리포트/방법론/사례/재측정표 |
| 위험 신호 | 측정 불명확/링크만 강조/과장/실행안 부재 |
| 판정 | 통과/보류/탈락과 추가 질문 |

```text
위험 신호:

최종 판단:
```

## 작성 예시

| 입력 항목 | 작성 예시 |
|---|---|
| 검증 항목 | 질문셋과 측정 방식 |
| 대행사 답변 | 월 1회 AI visibility 리포트 제공 |
| 증거 | 샘플 리포트에 source/citation 구분이 없음 |
| 위험 신호 | 백링크 수만 강조하고 질문 시장 설명이 부족 |
| 판정 | 보류, 질문셋과 지표 정의를 다시 요청한다 |

## 완료 기준

- 대행사가 무엇을 측정하고 무엇을 실행하는지 한 장으로 설명할 수 있습니다.
- 제안서의 강점과 위험 신호가 분리되어 있습니다.
- 질문셋, source/citation, 콘텐츠 액션, 재측정 계획이 이어져 있습니다.

## 흔한 질문

**Q. GEO 대행사는 SEO 대행사와 완전히 다른가요?**

기반 역량은 겹칩니다. 다만 GEO에서는 키워드 순위보다 질문셋, AI 답변, source/citation, 엔티티 합의, 재측정 기준이 더 중요해집니다.

**Q. 샘플 리포트가 없으면 탈락인가요?**

무조건 탈락은 아닙니다. 하지만 질문셋 예시, 측정 지표 정의, 결과 해석 방식은 최소한 확인해야 합니다.
