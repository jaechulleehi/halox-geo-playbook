## 2주차: Fan-out 질문맵과 콘텐츠 갭

2주차는 1주차에서 고른 대표 질문을 하위 질문으로 확장하고, 질문별 콘텐츠 갭을 찾는 실습입니다. 질문을 많이 만드는 것이 아니라 사업적으로 중요한 질문에서 답변 재료가 있는지 확인하는 것이 목표입니다.

좋은 Fan-out 질문맵은 3주차 리라이트 대상을 자동으로 만들어줍니다. 어떤 질문은 신규 콘텐츠가 필요하고, 어떤 질문은 기존 페이지의 첫 답변/표/FAQ를 고치면 됩니다.

## 2주차 목표

| 목표 | 설명 | 산출물 |
|---|---|---|
| 질문 확장 | 대표 질문을 하위 질문으로 나눈다 | Seed별 Fan-out 질문 |
| 질문 분류 | 구매/비교/문제해결/리스크/브랜드 질문으로 나눈다 | 질문 유형표 |
| 콘텐츠 갭 찾기 | 현재 콘텐츠가 답하지 못하는 부분을 찾는다 | 갭 리스트 |
| 리라이트 후보 선정 | 3주차에 고칠 페이지를 고른다 | 우선순위 1~3개 |

## 따라 해보는 순서

| 단계 | 할 일 | 확인할 것 |
|---|---|---|
| 1 | 1주차 기준선 공유 | “빠진 질문을 이번 단계에서 실행 대상으로 바꿉니다.” |
| 2 | Fan-out 예시 보기 | 하나의 질문이 비교/가격/도입/리스크로 퍼지는 구조 |
| 3 | 대표 질문 3개 확장 | Seed별 10개 내외 질문 작성 |
| 4 | 현재 콘텐츠 매칭 | 답할 수 있는 페이지와 없는 페이지 분리 |
| 5 | 콘텐츠 갭 우선순위 선정 | 매출/신뢰/반복 질문 기준으로 정렬 |
| 6 | 3주차 리라이트 후보 확정 | 고칠 페이지 1개와 보강 근거 정하기 |

## 실습 정리표

| 입력 항목 | 작성 기준 |
|---|---|
| Seed 질문 | 1주차에서 고른 핵심 질문 |
| 확장 질문 | Fan-out 결과 |
| 질문 유형 | 구매/비교/문제해결/리스크/브랜드 |
| 현재 콘텐츠 | 답할 수 있는 자산 |
| 갭 | 부족한 근거/구조/출처 |
| 리라이트 후보 | 다음 단계 작업 페이지 |

## 콘텐츠 갭 분류표

| 갭 유형 | 의미 | 다음 액션 |
|---|---|---|
| 답변 없음 | 질문에 대응하는 페이지가 없다 | 신규 콘텐츠 작성 |
| 구조 약함 | 페이지는 있지만 첫 답변/표/FAQ가 약하다 | Answer-first 리라이트 |
| 근거 약함 | 주장만 있고 답변 근거(source)/화면 인용(citation) 후보가 부족하다 | 답변 근거 맵 보강 |
| 기술 문제 | 페이지가 AI 크롤러에게 잘 읽히지 않는다 | 06장 기술 점검으로 연결 |
| 경쟁사 우세 | 경쟁사가 더 자주 언급된다 | 비교 콘텐츠/외부 출처 분석 |

## 정리 양식

```text
Seed 질문 3~5개:
확장 질문 30~50개:
질문 유형별 분포:
현재 콘텐츠로 답할 수 있는 질문:
콘텐츠 갭 10개:
3주차 리라이트 후보 1~3개:
필요한 답변 근거(source)/화면 인용(citation) 후보:
```

## 작성 예시

| 입력 항목 | 작성 예시 |
|---|---|
| Seed 질문 | GEO 도구 추천 |
| 확장 질문 | 가격 비교/AI 화면 인용 측정/AI Overview 대응/B2B SaaS 도입 기준 |
| 현재 콘텐츠 | GEO 정의 글과 AVI 점수 글 |
| 갭 | 도구 비교표와 샘플 리포트 부족 |
| 리라이트 후보 | GEO 도구 검증 기준 글 |

## 피드백 기준

| 항목 | 좋은 산출물 | 보완이 필요한 산출물 |
|---|---|---|
| 질문 확장 | 같은 Seed에서 여러 의도 질문이 나온다 | 비슷한 질문만 반복된다 |
| 갭 판단 | 현재 콘텐츠와 질문이 연결되어 있다 | 막연히 “글 필요”라고만 적혀 있다 |
| 우선순위 | 다음 단계에 고칠 페이지가 분명하다 | 갭 리스트만 있고 실행 대상이 없다 |

## 완료 기준

- Seed 질문 3개 이상에서 확장 질문이 만들어졌습니다.
- 콘텐츠 갭이 신규/리라이트/source/기술로 분류되어 있습니다.
- 3주차에 고칠 콘텐츠 1개가 확정되어 있습니다.

## 흔한 질문

**Q. Fan-out 질문은 AI에게 시켜도 되나요?**

가능합니다. 다만 그대로 쓰지 말고 실제 고객 말투, 구매 단계, 브랜드 상황에 맞게 걸러야 합니다.

**Q. 콘텐츠 갭이 너무 많이 나오면 어떻게 하나요?**

전환 가능성, 반복 질문 여부, 경쟁사 우세 여부를 기준으로 1~3개만 먼저 고릅니다.
