## 2주차: Fan-out 질문맵과 콘텐츠 갭

![Fan-out 질문맵과 콘텐츠 갭 흐름](../assets/images/page-heroes/halox-geo-10-02-week2-fanout-gap-workshop-hero.png)

2주차는 1주차에서 고른 대표 질문을 하위 판단 질문으로 확장하고, 질문별 콘텐츠 갭을 찾는 단계입니다. 질문을 많이 만드는 것이 아니라 사업적으로 중요한 질문에서 답변 재료가 어디서 끊기는지 확인하는 것이 목표입니다.

좋은 Fan-out 질문맵은 3주차 리라이트 대상을 자동으로 만들어줍니다. 어떤 질문은 신규 콘텐츠가 필요하고, 어떤 질문은 기존 페이지의 첫 답변/표/FAQ를 고치면 됩니다. 어떤 질문은 콘텐츠보다 외부 출처, 기술 구조, 브랜드 정의가 먼저 필요할 수도 있습니다.

## 2주차 목표

| 목표 | 설명 | 산출물 |
|---|---|---|
| 질문 확장 | 대표 질문을 하위 질문으로 나눈다 | Seed별 Fan-out 질문 |
| 질문 분류 | 구매/비교/문제해결/리스크/브랜드 질문으로 나눈다 | 질문 유형표 |
| 콘텐츠 갭 찾기 | 현재 콘텐츠가 답하지 못하는 부분을 찾는다 | 갭 리스트 |
| 리라이트 후보 선정 | 3주차에 고칠 페이지를 고른다 | 우선순위 1~3개 |

## Fan-out을 왜 따로 보나

사람은 하나의 질문을 입력하지만, AI 답변은 보통 여러 하위 판단을 거쳐 만들어집니다. 예를 들어 “GEO 도구 추천해줘”라는 질문에는 도구 정의, 비교 기준, 가격, 기능, 신뢰도, 고객 유형, 대안, 리스크 같은 판단이 숨어 있습니다.

| 사용자가 입력한 질문 | AI가 내부적으로 확인할 수 있는 하위 판단 | 필요한 답변 재료 |
|---|---|---|
| GEO 도구 추천해줘 | GEO 도구의 정의는 무엇인가? | 카테고리 정의/용어집 |
| GEO 도구 추천해줘 | 어떤 지표로 비교해야 하나? | mention/source/citation/경쟁사 비교표 |
| GEO 도구 추천해줘 | 어떤 브랜드가 후보군인가? | 공식 사이트/리뷰/외부 목록/사례 |
| GEO 도구 추천해줘 | 누구에게 맞는가? | 업종별 사용 기준/B2B SaaS/커머스/로컬 예시 |
| GEO 도구 추천해줘 | 과장된 약속은 어떻게 거르나? | 제안서 검증표/리포트 샘플/주의 문장 |

이 하위 판단을 보지 않으면 콘텐츠 갭도 “글이 부족하다” 정도로만 보입니다. 2주차의 핵심은 부족한 글 제목을 찾는 것이 아니라, AI 답변이 필요로 하는 판단 재료를 찾는 것입니다.

## 질문 확장과 AI Fan-out은 다르다

2주차에서 헷갈리기 쉬운 부분이 있습니다. 우리가 질문을 늘리는 일과 AI가 답변을 만들기 위해 내부적으로 질문을 쪼개는 일은 연결되어 있지만 같은 말은 아닙니다.

| 구분 | 의미 | 2주차에서 하는 일 |
|---|---|---|
| 사용자 질문 확장 | SEO 키워드, 고객 질문, PAA, 연관 검색을 보고 측정 질문을 넓히는 일 | 기준선 측정용 질문셋 확장 |
| AI Fan-out | AI가 하나의 질문에 답하기 위해 정의/비교/추천/검증/source/URL 확인을 나누는 현상 | 답변 재료가 필요한 판단 노드 추정 |
| 콘텐츠 갭 분석 | Fan-out 노드별로 현재 콘텐츠/source/기술이 부족한 부분을 찾는 일 | 3주차 실행 후보 결정 |

자세한 개념은 [03-01. Fan-out은 AI가 질문을 어떻게 쪼개는가](https://wikidocs.net/346344)에서 먼저 확인할 수 있습니다.

## 따라 해보는 순서

| 단계 | 할 일 | 확인할 것 |
|---|---|---|
| 1 | 1주차 대표 질문 3개 선택 | mention이 약하거나 경쟁사 우세가 뚜렷한 질문을 고른다 |
| 2 | 질문 안의 하위 판단 쓰기 | 정의/비교/가격/신뢰/리스크/도입 조건 분리 |
| 3 | Seed별 확장 질문 10개 작성 | 같은 표현 반복이 아니라 다른 판단 기준을 만든다 |
| 4 | 현재 콘텐츠와 매칭 | 답할 수 있는 페이지와 없는 페이지를 나눈다 |
| 5 | 갭 유형 분류 | 신규/리라이트/source/기술/메시지 갭으로 나눈다 |
| 6 | 3주차 후보 선정 | 고칠 페이지 1~3개와 필요한 근거를 정한다 |

| GEO 도구 추천해줘 | GEO 도구는 어떤 지표를 봐야 하나? | 비교형 | 09장 리포트 기준 | 비교표 부족 | 도구 비교표 보강 |

| GEO 도구 추천해줘 | 과장된 대행사 제안은 어떻게 거르나? | 리스크형 | 09-04 | 체크리스트는 있음/source 약함 | 검증 질문 보강 |
| GEO 도구 추천해줘 | 실제 리포트는 어떤 모양이어야 하나? | 실행형 | 10-04 | 샘플 부족 | 1페이지 리포트 예시 작성 |

## 콘텐츠 갭 분류표

| 갭 유형 | 의미 | 확인 질문 | 다음 액션 |
|---|---|---|---|
| 답변 없음 | 질문에 대응하는 페이지가 없다 | 이 질문에 바로 답하는 URL이 있는가? | 신규 콘텐츠 작성 |
| 구조 약함 | 페이지는 있지만 첫 답변/표/FAQ가 약하다 | 첫 5문장만 읽어도 답이 되는가? | Answer-first 리라이트 |
| 근거 약함 | 주장만 있고 답변 근거(source)/화면 인용(citation) 후보가 부족하다 | AI가 신뢰할 내부/외부 근거가 있는가? | source map 보강 |
| 기술 문제 | 페이지가 AI 크롤러에게 잘 읽히지 않는다 | HTML/링크/schema/canonical이 확인되는가? | 06장 기술 점검으로 연결 |
| 메시지 불일치 | 공식 소개/외부 프로필/콘텐츠 문장이 다르다 | 브랜드 한 줄 정의가 일관적인가? | 엔터티 문장 정렬 |
| 경쟁사 우세 | 경쟁사가 더 자주 언급된다 | 경쟁사는 어떤 질문에서 선택되는가? | 비교 콘텐츠/외부 출처 분석 |

## 우선순위 점수표

갭이 너무 많으면 실행이 멈춥니다. 아래 기준으로 1~3개만 3주차 후보로 넘깁니다.

| 기준 | 1점 | 3점 | 5점 |
|---|---|---|---|
| 전환 가능성 | 정보 탐색 수준 | 비교/검토 단계 | 구매/상담/도입 직전 |
| 반복성 | 일회성 질문 | 가끔 나오는 질문 | 매달 반복되는 핵심 질문 |
| 경쟁사 우세 | 경쟁사도 약함 | 일부 경쟁사 우세 | 특정 경쟁사가 반복 추천됨 |
| 수정 난이도 | 새 시스템 필요 | 기존 페이지 큰 수정 | 첫 문단/표/FAQ 수정 가능 |
| 근거 확보 가능성 | 출처 없음 | 내부 근거 있음 | 내부/외부 근거 모두 있음 |

합계가 높고 수정 난이도가 낮은 항목부터 고릅니다. “중요하지만 오래 걸리는 항목”은 4주차 30일 계획으로 넘기고, 3주차에는 실험 가능한 한 페이지를 먼저 고르는 편이 좋습니다.

## 정리 양식

```text
Seed 질문 3~5개:
확장 질문 30~50개:
질문 유형별 분포:
현재 콘텐츠로 답할 수 있는 질문:
콘텐츠 갭 10개:
갭 유형 분류:
우선순위 점수:
3주차 리라이트 후보 1~3개:
필요한 답변 근거(source)/화면 인용(citation) 후보:
기술 점검이 필요한 URL:
```

## 작성 예시

| 입력 항목 | 작성 예시 |
|---|---|
| Seed 질문 | GEO 도구 추천 |
| 확장 질문 | 가격 비교/AI 화면 인용 측정/AI Overview 대응/B2B SaaS 도입 기준 |
| 현재 콘텐츠 | GEO 정의 글과 AVI 점수 글 |
| 갭 | 도구 비교표와 샘플 리포트 부족 |
| 우선순위 | 전환 가능성 5점, 반복성 5점, 수정 난이도 4점 |
| 리라이트 후보 | GEO 도구 검증 기준 글 |

## 완료 기준

- Seed 질문 3개 이상에서 확장 질문이 만들어졌습니다.
- 질문 확장과 AI Fan-out을 구분해 기록했습니다.
- 콘텐츠 갭이 신규/리라이트/source/기술/메시지로 분류되어 있습니다.
- 3주차에 고칠 콘텐츠 1개와 필요한 근거가 확정되어 있습니다.

## 흔한 질문

**Q. Fan-out 질문은 AI에게 시켜도 되나요?**

가능합니다. 다만 그대로 쓰지 말고 실제 고객 말투, 구매 단계, 브랜드 상황에 맞게 걸러야 합니다. AI가 만든 질문은 초안이고, 최종 질문셋은 사람이 시장/고객/전환 기준으로 정리해야 합니다.

**Q. 콘텐츠 갭이 너무 많이 나오면 어떻게 하나요?**

전환 가능성, 반복 질문 여부, 경쟁사 우세 여부를 기준으로 1~3개만 먼저 고릅니다. 나머지는 4주차 30일 액션 플랜의 후보로 남깁니다.

**Q. PAA나 연관 검색어가 곧 Fan-out인가요?**

아닙니다. PAA와 연관 검색어는 사용자 질문 확장의 단서입니다. AI Fan-out은 답변을 만들기 위한 내부 판단 노드를 추정하는 관점입니다. 둘을 함께 보면 좋지만 같은 것으로 쓰면 안 됩니다.
