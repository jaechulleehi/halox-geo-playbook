## 3주차: 콘텐츠 리라이트와 출처 설계

![3주차 콘텐츠 리라이트와 source 설계](../assets/images/page-heroes/halox-geo-10-03-week3-content-source-workshop-hero.png)

3주차의 목표는 2주차에서 찾은 갭을 실제 페이지 수정과 source 설계로 바꾸는 것입니다. 첫 문단을 Answer-first로 고치고, 비교 기준과 근거를 보강하며, 필요한 외부 출처를 정리합니다.

GEO 리라이트는 문장 교정이 아닙니다. 질문, 답변, 근거, 내부 링크, 외부 source, 재측정 질문이 함께 연결되어야 합니다.

[TOC]

## 먼저 볼 기준

| 기준 | 읽는 법 |
|---|---|
| 콘텐츠 | 첫 답변과 비교 기준을 앞에 둔다 |
| source | 공식/외부 근거를 나눠 보강한다 |
| 기술 | citation 후보 URL의 발견/읽기 조건을 확인한다 |

## 실행 흐름

1. 대표 질문을 정한다.
2. 현재 AI 답변에서 mention/source/citation을 나눠 본다.
3. 경쟁 브랜드나 반복 URL이 어떤 이유로 등장하는지 확인한다.
4. 우리 공식 페이지, 외부 출처, 기술 조건 중 먼저 고칠 곳을 고른다.
5. 같은 질문군으로 30일 뒤 다시 본다.

![Answer-first와 출처 설계를 함께 고치는 흐름](../assets/images/body-figures/halox-geo-10-03-week3-answer-first-source-design-flow-codex-only.png)

*Answer-first와 출처 설계를 함께 고치는 흐름*

## 3주차 예시

AcmeGEO는 제품 페이지 첫 문단을 “브랜드 언급률, source, citation을 분리해 보여주는 GEO 리포트 도구”로 고치고, 리포트 예시 페이지와 외부 해설 글을 연결합니다.

## 정리 양식

```text
수정 URL:
대표 질문:
새 첫 답변:
보강 근거:
외부 source:
재측정 질문:
```
