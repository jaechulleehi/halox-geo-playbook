## 3주차: 콘텐츠 리라이트와 출처 설계

3주차는 2주차에서 고른 콘텐츠 하나를 Answer-first 구조로 고치고, source/citation 후보를 함께 설계하는 시간입니다. GEO 콘텐츠는 글만 고치는 작업이 아니라 AI가 답변에 쓸 수 있는 근거와 구조를 정리하는 작업입니다.

콘텐츠만 고치면 자사 주장에 머물 수 있고, 외부 신호만 만들면 답변 재료가 약할 수 있습니다. 둘을 함께 설계해야 합니다.

## 3주차 목표

| 목표 | 설명 | 산출물 |
|---|---|---|
| 리라이트 대상 확정 | 2주차 갭에서 고칠 페이지를 고른다 | 대상 URL/문서 |
| 첫 답변 수정 | 질문에 바로 답하는 opening을 쓴다 | Answer-first 첫 문단 |
| 구조 보강 | FAQ/표/schema 후보를 정리한다 | 구조화 초안 |
| 출처 설계 | 내부/외부 source 후보를 분리한다 | source map |
| 재측정 질문 확정 | 수정 후 다시 볼 질문을 정한다 | 재측정 질문셋 |

## 90분 진행안

| 시간 | 진행 | 읽으면서 확인할 것 |
|---|---|---|
| 0~10분 | 2주차 갭과 리라이트 후보 공유 | “오늘은 글 하나를 실제로 고칩니다.” |
| 10~25분 | Answer-first 예시 보기 | 결론/기준/근거/FAQ가 앞에 오는 구조 |
| 25~50분 | 첫 문단과 표/FAQ 초안 작성 | AI 답변 재료가 되는 문장 만들기 |
| 50~65분 | source/citation 후보 정리 | 자사 페이지와 외부 근거 분리 |
| 65~80분 | 2인 교차 검수 | 첫 문단만 읽어도 답이 되는지 확인 |
| 80~90분 | 발행/재측정 체크 확정 | 4주차 실행 리포트에 넣을 액션 정리 |

## 실습 워크시트

| 입력 항목 | 작성 기준 |
|---|---|
| 리라이트 대상 | URL 또는 문서명 |
| 목표 질문 | AI 답변에서 잡고 싶은 질문 |
| 첫 답변 | 수정한 opening |
| 추가 구조 | FAQ/표/schema/비교 기준 |
| 출처 후보 | 내부/외부 근거 |
| 검수 | AI가 인용할 만한 문장 |

## 리라이트 전후 점검표

| 항목 | Before | After |
|---|---|---|
| 첫 문단 | 배경 설명이 길다 | 질문에 바로 답한다 |
| 구조 | 긴 문단 중심 | 표/FAQ/기준으로 분리 |
| 근거 | 자사 주장 중심 | 내부/외부 source가 분리됨 |
| 인용 가능성 | 문장 단위로 가져가기 어렵다 | 정의/기준/비교 문장이 명확함 |
| 다음 액션 | 글 발행으로 끝남 | 재측정 질문과 날짜가 있음 |

## 정리 양식

```text
대상 콘텐츠:
목표 질문:
바꾼 첫 문단:
추가 FAQ:
추가 표/비교 기준:
source 후보 5개:
재측정 질문:
발행 전 체크:
```

## 작성 예시

| 입력 항목 | 작성 예시 |
|---|---|
| 리라이트 대상 | GEO 콘텐츠 구조화 가이드 |
| 목표 질문 | AI에게 인용되는 GEO 글은 어떻게 써야 하나? |
| 첫 답변 | AI가 인용하기 쉬운 GEO 글은 결론, 기준, 근거, FAQ가 앞에 있는 글입니다. |
| 출처 후보 | AVI 점수 가이드, citation 설명 글, 외부 업계 리포트 |
| 검수 | 첫 문단과 표만 읽어도 답변 재료가 된다 |

## 피드백 기준

| 항목 | 좋은 산출물 | 보완이 필요한 산출물 |
|---|---|---|
| 첫 답변 | 질문에 2~4문장 안에 답한다 | 배경 설명만 길고 결론이 늦다 |
| 구조 | 표/FAQ/기준이 답변을 돕는다 | 소제목만 있고 판단 기준이 없다 |
| source map | 내부/외부 출처가 분리되어 있다 | 참고 링크가 무작위로 붙어 있다 |

## 완료 기준

- 리라이트 대상 콘텐츠 1개가 정해졌습니다.
- 첫 문단, 표/FAQ, source 후보, 재측정 질문이 함께 작성되었습니다.
- 발행 후 4주차 리포트에 넣을 실행 로그가 남아 있습니다.

## 흔한 질문

**Q. 기존 글을 완전히 새로 써야 하나요?**

아닙니다. 먼저 첫 문단, 핵심 표, FAQ, source 후보를 고치는 것만으로도 1차 실험이 됩니다.

**Q. 외부 출처는 꼭 만들어야 하나요?**

모든 페이지에 필요한 것은 아닙니다. 다만 AI가 신뢰할 근거가 자사 페이지 안에만 갇혀 있으면 citation 안정성이 약해질 수 있습니다.
