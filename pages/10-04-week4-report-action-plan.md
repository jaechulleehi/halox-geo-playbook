## 4주차 GEO 실행 리포트와 30일 액션 플랜

![4주차 GEO 실행 리포트](../assets/images/page-heroes/halox-geo-10-04-week4-report-action-plan-hero.png)

4주차의 목표는 한 달 동안 확인한 기준선, Fan-out 갭, 콘텐츠/source 수정 결과를 하나의 GEO 실행 리포트로 묶는 것입니다. 리포트는 지식 정리가 아니라 다음 30일 행동을 정하는 문서입니다.

좋은 실행 리포트는 “무엇을 배웠는가”보다 “다음 달 무엇을 고칠 것인가”를 분명히 합니다. 질문군별 변화, 경쟁 URL, 남은 리스크, 담당 액션이 보여야 합니다.

`AcmeGEO`라는 이름은 설명을 위한 가상 기업명이며, 실제 고객 사례가 아닙니다.

## 먼저 볼 기준

| 기준 | 읽는 법 |
|---|---|
| 요약 | 5줄 안에 상태/원인/액션을 말한다 |
| 우선순위 | 영향도와 실행 가능성을 함께 본다 |
| 반복 | 다음 측정 질문과 날짜를 고정한다 |

## 실행 흐름

1. 1주차와 같은 질문셋으로 mention/source/citation을 다시 측정한다.
2. 좋아진 질문, 그대로인 질문, 더 나빠진 질문을 나눈다.
3. 변화 원인을 콘텐츠 수정, source 보강, 기술 이슈, 경쟁 변화로 설명한다.
4. 다음 30일 액션을 3개 이하로 줄이고 담당자와 산출물을 붙인다.
5. 월간 리포트 문장으로 남겨 다음 사이클의 기준선으로 쓴다.

*30일 액션 플랜으로 마무리하는 흐름*

## 4주차 예시

AcmeGEO는 추천형 질문에서 mention은 늘었지만 citation은 아직 약하다고 정리합니다. 다음 30일 액션은 리포트 예시 URL 보강, 외부 비교 글 확보, 같은 질문군 재측정입니다.

## 작성 예시와 완료 기준

| 항목 | 예시 |
|---|---|
| 이번 달 변화 | 추천형 질문 mention 8/30 → 14/30, citation 3/30 → 6/30 |
| 효과가 있었던 수정 | 리포트 예시 페이지 첫 문단, 지표 표, 내부 링크 보강 |
| 남은 문제 | 경쟁사 glossary가 source로 반복됨 |
| 다음 30일 | glossary 보강, 외부 디렉터리 설명 수정, 기술 점검 |

완료 기준은 “이번 달 리포트 작성”이 아닙니다. 다음 30일 동안 누가 무엇을 고칠지 결정되어야 합니다.
