## PR/마케팅 에이전시 GEO 서비스 설계

![PR 마케팅 에이전시 GEO 서비스 패키지](../assets/images/page-heroes/halox-geo-90-01-pr-agency-geo-service-case-hero.png)

PR/마케팅 에이전시가 GEO를 서비스로 만들 때 가장 위험한 문장은 “AI 검색에 노출되게 해드립니다”입니다. 고객은 성과를 기대하지만, 팀은 무엇을 측정하고 무엇을 실행할지 합의하지 못한 채 보도자료, 블로그, 리포트가 뒤섞이기 쉽습니다.

좋은 GEO 서비스는 노출 보장이 아니라 운영 범위를 제안합니다. 질문셋을 정하고, 현재 mention/source/citation을 측정하고, 공식/외부 출처의 공백을 찾고, 콘텐츠/뉴스룸/기술 수정으로 나눈 뒤, 월간 리포트로 다시 확인합니다.

## 가상 기업 AcmePR의 시작 상황

AcmePR은 B2B SaaS 고객사에게 GEO 운영 패키지를 제안하려고 합니다. 고객사는 “우리 브랜드가 ChatGPT와 Perplexity에서 잘 보이게 해달라”고 말하지만, 실제 문제는 더 좁습니다. 브랜드명 질문에서는 이미 언급되지만, “보안 SaaS 추천”, “대기업용 접근제어 솔루션 비교” 같은 비브랜드/비교 질문에서는 경쟁사와 외부 리뷰가 더 자주 나옵니다.

AcmePR이 바로 기사 배포량을 늘리면 고객은 잠시 안심할 수 있지만, 다음 달 리포트에서 무엇이 좋아졌는지 설명하기 어렵습니다. 먼저 기준선을 잡아야 합니다.

| 진단 항목 | 확인한 신호 | 서비스 범위로 바꾸는 법 |
|---|---|---|
| 질문셋 | 브랜드/비브랜드/비교/구매 검토 질문 분리 | 월간 고정 질문 30개 |
| mention | 브랜드가 언급되는 질문과 빠지는 질문 | 후보군 진입 관리 |
| source/citation | 공식 뉴스룸, 외부 기사, 경쟁사 URL | 출처 보강 작업 |
| 실행 | 뉴스룸, 비교 FAQ, 외부 프로필, 기술 이슈 | 담당자와 산출물 분리 |

## 제안서에 넣을 것과 빼야 할 것

제안서에는 “AI가 좋아하는 콘텐츠” 같은 표현보다 측정 조건을 넣어야 합니다. 어떤 AI 엔진, 어떤 질문군, 어떤 기간, 어떤 URL을 기준으로 볼지 정하지 않으면 고객은 결과를 임의로 해석합니다.

반대로 “모든 AI 답변에서 상위 노출” 같은 문장은 빼야 합니다. GEO는 검색 광고처럼 고정 순위를 사는 일이 아니고, AI 답변은 질문, 모델, 시점, 출처 상태에 따라 달라집니다.

## HaloX 리포트로 운영하는 흐름

AcmePR은 먼저 프롬프트 분석으로 질문군별 답변을 봅니다. 인용 추적에서는 공식 뉴스룸, 외부 기사, 경쟁사 URL이 어떤 질문에서 citation으로 쓰이는지 분리합니다. 사이트 진단은 고객사의 뉴스룸/자료실/제품 페이지가 기술적으로 읽히는지 확인합니다.

그다음 전략맵에서 이번 달 실행 클러스터를 고릅니다. 예를 들어 `보안 비교`, `도입 사례`, `가격/범위`, `규제 대응` 중 하나를 우선순위로 정하고, 콘텐츠 제작이나 PR 출처 보강으로 넘깁니다. 주간 리포트는 점수표가 아니라 “이번 주 어떤 질문에서 공식 citation이 회복됐는가”를 설명합니다.

![PR 에이전시 GEO 서비스 패키지 맵](../assets/images/body-figures/halox-geo-90-01-pr-agency-geo-service-package-map-codex-only.png)

*PR 에이전시 GEO 서비스는 기사 배포가 아니라 질문셋, 출처, 실행 범위, 재측정 리포트를 묶는 운영 패키지다.*

## 4주 운영 예시

| 주차 | 할 일 | 산출물 |
|---|---|---|
| 1주차 | 질문셋과 기준선 측정 | mention/source/citation 진단표 |
| 2주차 | 뉴스룸/외부 출처/경쟁 URL 분석 | 출처 공백 지도 |
| 3주차 | 비교 FAQ, 리포트 예시, 프로필 설명 보강 | 콘텐츠/PR 수정안 |
| 4주차 | 같은 질문으로 재측정 | 월간 리포트와 다음 액션 |

## 고객에게 말할 문장

“이번 패키지는 AI 검색 노출을 보장하는 상품이 아니라, 고객사가 어떤 질문에서 빠지고 어떤 출처가 대신 인용되는지 확인한 뒤 공식 근거와 외부 신뢰 신호를 보강하는 월간 운영입니다. 첫 달 성공 기준은 전체 점수 상승보다 비브랜드 비교 질문에서 공식 URL citation이 생기는지입니다.”

## 정리 양식

```text
고객 업종:
월간 고정 질문셋:
현재 빠지는 질문군:
반복 경쟁 브랜드/URL:
공식 뉴스룸/자료실 상태:
보강할 외부 출처:
이번 달 실행 범위:
다음 리포트 날짜:
```
