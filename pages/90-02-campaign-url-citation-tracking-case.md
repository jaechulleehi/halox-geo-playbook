## 캠페인 URL 인용 추적: 한 번 언급과 반복 인용의 차이

캠페인 페이지나 PR URL이 AI 답변에 한 번 등장했다고 해서 곧바로 GEO 성과라고 보기는 어렵습니다. 중요한 것은 같은 질문군에서 반복적으로 답변 근거가 되는지, 그리고 사용자가 볼 수 있는 화면 인용으로 드러나는지입니다.

이 사례는 캠페인 성과를 `노출됐다/안 됐다`가 아니라 `어떤 질문에서 어떤 URL이 반복 인용되는가`로 읽는 방법을 다룹니다.

## 상황

브랜드는 캠페인 페이지, 이벤트 랜딩, 보도자료, 제휴 콘텐츠, 블로그 글을 많이 만듭니다. 하지만 AI 검색에서는 URL 하나가 잠깐 언급되는 것과, 여러 질문에서 안정적으로 인용되는 것이 다릅니다.

캠페인 URL 추적은 단기 캠페인을 GEO 관점으로 해석하는 좋은 훈련입니다.

## SEO 키워드 관점

| SEO 키워드 | 사용자의 실제 의도 | 연결할 GEO 질문 |
|---|---|---|
| 캠페인 성과 측정 | URL 노출과 전환 확인 | AI 답변에 캠페인 URL이 등장하는가? |
| 브랜드 캠페인 SEO | 검색 결과에서 캠페인 발견 | 검색 TOP10과 AI 답변이 겹치는가? |
| AI 검색 인용 | AI 답변의 출처 확인 | 화면 인용에 우리 URL이 보이는가? |
| PR URL 추적 | 보도/캠페인의 재사용 확인 | 어떤 출처가 반복적으로 선택되는가? |

## AI 질문/GEO 관점

| 질문군 | 예시 | 측정 포인트 |
|---|---|---|
| 캠페인 직접 질문 | OO 캠페인은 무엇인가? | 캠페인 URL mention/source |
| 카테고리 질문 | 친환경 패키징 사례는? | 캠페인이 사례 후보로 들어가는지 |
| 비교 질문 | A 캠페인과 B 캠페인의 차이는? | 경쟁 캠페인과 함께 비교되는지 |
| 검증 질문 | 이 캠페인은 실제 효과가 있었나? | 외부 기사/리포트 citation |

## 자주 막히는 지점

가장 흔한 오해는 검색 결과 순위가 높으면 AI 답변에서도 인용될 것이라고 보는 것입니다. 검색 결과에 있어도 AI 답변에 쓰이지 않을 수 있고, 반대로 AI 답변의 배경 근거로 쓰이지만 화면 인용에는 드러나지 않을 수도 있습니다.

따라서 `검색 노출`, `답변 근거`, `화면 인용`을 분리해 봐야 합니다.

## URL 인용 추적표

| 측정 항목 | 기록 방법 | 판단 기준 |
|---|---|---|
| 질문 원문 | 반복 측정할 질문 그대로 저장 | 질문이 바뀌면 결과 비교 불가 |
| 검색 TOP10 | 일반 검색 결과 URL 기록 | 기존 SEO 기반 확인 |
| 답변 내 언급 | 브랜드/캠페인명이 나오는지 확인 | mention 여부 |
| 답변 근거 | 답변 내용이 어떤 URL을 반영하는지 추정 | source 후보 |
| 화면 인용 | 사용자에게 보이는 링크 기록 | citation 안정성 |
| 경쟁 URL | 함께 등장하는 경쟁 캠페인 기록 | 비교 문맥 |

## 실행 순서

1. 캠페인 핵심 키워드와 질문군을 15~20개 정합니다.
2. 캠페인 시작 전 기준선을 측정합니다.
3. 캠페인 페이지/보도자료/블로그/외부 기사 URL을 분리해 기록합니다.
4. 캠페인 중간과 종료 후 같은 질문을 다시 측정합니다.
5. 한 번 나온 URL보다 반복 인용되는 URL을 찾습니다.
6. 반복 인용되지 않는다면 본문 구조, 외부 출처, 내부 링크를 보강합니다.

## 좋은 예와 나쁜 예

| 나쁜 예 | 좋은 예 |
|---|---|
| AI 답변에 한 번 나왔으니 성공 | 같은 질문군에서 반복 인용되는지 재측정 |
| 캠페인 페이지 하나만 추적 | 보도자료/외부 기사/블로그/리뷰 URL을 분리 추적 |
| 검색 순위만 보고 판단 | 검색 TOP10, 답변 근거, 화면 인용을 나눠 판단 |
